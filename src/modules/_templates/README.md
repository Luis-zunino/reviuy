# Templates De Modulos

Estos templates sirven como punto de partida para nuevos modulos o nuevos casos de uso dentro de modules.

## Objetivo

Estandarizar la forma de crear:

- use cases
- query services
- repositories

## Uso

1. copiar el archivo base correspondiente
2. renombrarlo segun el caso de uso o query real
3. reemplazar nombres y contratos por el dominio concreto
4. implementar primero application y domain antes que presentation
